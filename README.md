# Used Car Sales Interface

A filterable used car dealership interface built with vanilla HTML, CSS, and JavaScript — deployed via GitHub Pages.

---

## Overview

This project is a front-end web application that simulates a used car dealership listing page. Users can browse available inventory and narrow down results using a set of real-time filters — no page reloads required.

Built as part of **CSI 3150 – Web Apps Programming I** at Oakland University (Winter 2026).

---

## Features

- **Car listings** generated from a JavaScript data file
- **Multi-field filtering** with instant results:
  - Year range (min / max)
  - Make (select one or more)
  - Max mileage threshold
  - Price range (min / max)
  - Color (select one or more)
- **No-results state** with a user-friendly prompt to adjust filters
- Semantic HTML throughout for accessibility and structure
- Clean, responsive card-based layout inspired by real dealership websites
